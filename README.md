## Project Overview
This project provides a deep dive into the **Global Super Store** dataset (2011-2014). The dashboard tracks performance across various dimensions to help stakeholders understand sales drivers and profit margins.

## Key Insights
* **Total Performance**: Analyzed data for **780 customers**, reaching **$619K in Sales** and **$71K in Profit**.
* **Sales by Category**: **Technology** is the leading category with $237K in sales.
* **Profitability**: Technology also yields the highest profit ($33K), followed by Office Supplies ($27K).
* **Customer Segments**: The **Consumer** segment is the largest contributor, accounting for 53% of sales and 57% of profit.
* **Global Reach**: Interactive mapping shows high activity across North America, Europe, and Asia.

## Features
* **Yearly Trends**: Time-series analysis from 2011 to 2014.
* **Drill-Through**: Users can right-click on countries for deeper data exploration.
* **Interactive Filters**: Dynamic filtering by Year and Segment.
